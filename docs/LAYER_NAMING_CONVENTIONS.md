# Layer Naming Conventions

These names should stay aligned with the existing ArmourCore Affinity template and downstream Insert Builder logic.

## Core layers
- `Outside Edge`
- `Badge Pockets`
- `Laser Labels`
- `Notes`
- `Reference & Scaling`
- `Template Image`

## Depth layers
Format:
- `<number>mm`

Examples:
- `25mm`
- `30mm`
- `35mm`
- `40mm`

## Through-cut layer
- `THRU`
